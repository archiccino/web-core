Introduction

Welcome to Archiccino Linux, a Linux distribution crafted specifically for programmers. Designed to offer an out-of-the-box, feature-rich experience, Archiccino is perfect for anyone who values efficiency, functionality, and customizability.

I created Archiccino Linux primarily for myself, as I wanted a distribution that aligns with my personal development workflow. While this distro reflects my preferences, it’s flexible enough to adapt to anyone’s needs with its unique Flavours feature.

Features

Archiccino Linux comes packed with tools and features to support developers from the get-go. Here are some highlights:

  • Pre-installed Development Tools: Includes popular compilers, interpreters, and IDEs like VS Code, Vim, etc.
  • Feature-rich Terminal: Zsh shell with oh-my-posh, nerd fonts, and extensive plugins for maximum productivity.
  • Customised Workflows: Pre-configured hyprland desktop environment optimized for coding.
  • Seamless Software Management: Pacman, AUR and Flatpak support through a single cli app written in rust for accessing thousands of packages.

Flavours

One of the unique features of Archiccino Linux is its Flavours. These are pre-defined configurations that allow you to tweak the system to suit your specific requirements. For example:

Web Developer Flavour
Includes tools like Node.js, Docker, and frontend build systems.
Data Scientist Flavour
Pre-installed Python libraries, Jupyter Notebook, and RStudio.
System Programmer Flavour
Optimized with C/C++ toolchains, debugging tools, and kernel headers.

You can also create your own flavour by defining a simple configuration file and running the flavour setup script.
